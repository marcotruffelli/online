# 💻 Simple Web Project - School Group Project

This repository contains a **simple web project** created as part of a school group assignment.  
The task was to develop a website about an **IT-related topic**. The project is basic, built only with **HTML and CSS**, and is **not responsive**, as it was made at the beginning of our web development learning journey.

---

## 📝 Overview

The website was designed to:

- Present information on a chosen IT topic  
- Practice **HTML and CSS skills**  
- Work collaboratively in a group environment
